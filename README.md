# Menstrual Tracking System

A simple command-line application developed in Python that helps users predict their next menstrual cycle based on the first day of their last period and their average cycle length.

## Features
- Predicts the next period date
- Calculates the ovulation day
- Displays the fertile window
- Uses Python's datetime module for accurate date calculations
- Easy-to-use command-line interface

## Technologies Used
- Python
- datetime module

## How It Works
1. Enter your name.
2. Enter the first day of your last period.
3. Enter your average cycle length.
4. Enter your period duration.
5. The application predicts:
   - Next period date
   - Ovulation day
   - Fertile window
